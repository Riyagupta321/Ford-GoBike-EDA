# Ford GoBike EDA

Exploratory Data Analysis on 12 months (2018) of Ford GoBike (Bay Area bike-share) trip data.

## Project Overview
- 1.85M+ trip records analyzed
- Data cleaning, feature engineering, and 15 visualizations
- Univariate, Bivariate, and Multivariate analysis
- Business recommendations based on rider behavior patterns

## Files
- `Ford_GoBike_EDA.ipynb` - Main analysis notebook

## Key Findings
- Subscribers (~85% of trips) show clear weekday commute patterns
- Customers take longer, farther trips concentrated on weekends
- Peak usage hours: 8 AM and 5-6 PM on weekdays
